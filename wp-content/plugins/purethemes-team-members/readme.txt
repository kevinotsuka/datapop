== Description ==

Providing a convenient way to show your team member in any full width templates or pages. Custom post types, drag and drop re-arrange system, shortcode activation, Easiest admin setting panel.

ACTIVATION GUIDE:

- Use [pure_all_members] shortcode in Editor.

***See More Information of settings in the Installation Tab

Links:- <a href="http://puretheme.zinan.me/plugins/puretheme-team-members/">Demo Plugin Preview</a>


== Installation ==

1. Upload `pure-theme-team-members` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Text Editor of any page or custom full width template by [pure_all_members] shortcode.
3. Use "Member Setting" to make change of the default styles. (Please Press "Save Changes")
4. Under "Member Details" section new member details need to be added.
5. Use "Re - Order" to re-arrange the member section. 

****** Important Note *******
At present this plugin is produced for full width large template (1170px width will be perfect). Max 4 members can be added.
